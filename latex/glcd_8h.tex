Update bounding box.

The bounding box defines a rectangle in which needs to be refreshed next time \hyperlink{group___controllers_ga0df6c0642fff31b764298958eda56efb}{glcd\+\_\+write()} is called. \hyperlink{group___controllers_ga0df6c0642fff31b764298958eda56efb}{glcd\+\_\+write()} only writes to those pixels inside the bounding box plus any surrounding pixels which are required according to the bank/column write method of the controller.

Define a rectangle here, and it will be {\itshape added} to the existing bounding box.
